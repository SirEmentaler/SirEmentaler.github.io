Comments

Comments are a language feature most commonly used for placing documentation within the source code. Any content placed inside of comments is completely ignored by the compiler. This allows a script to contain notes that are readable to humans but don't affect behavior of the script.

Syntax

/* comment */(1)
// comment(2)
  1. Multi-line comment
  2. Single-line comment

Example

/* Multi-line comments
may span
multiple lines */

/* or just one */

// Single-line comments can only
// contain one line each

void onLevelLoad() {
	// jjDebug("Code inside comments doesn't run");
	jjDebug("Other code runs as usual");
}
