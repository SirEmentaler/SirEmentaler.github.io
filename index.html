Additionally, there exist identifiers that are not keywords but gain special functionality in certain contexts:

final
from
 get
override
 set
shared
 super
this
and
Alternative representation of the && (logical and) operator.
auto
Auto specifier; specifies that the type of the variable that is being defined will be automatically deduced from its initializer.
bool
Boolean type.
break
Break statement; causes the enclosing for, while or do-while loop or switch statement to terminate.
do
do-while loop declaration.
double
Double-precision floating-point format number type.
false
False boolean literal.
float
Single-precision floating-point format number type.
for
for loop declaration.
int
32-bit signed integer type.
int8
int16
int32
int64
n-bit signed integer type.
is
Identity operator.
not
Alternative representation of the ! (negation) operator.
null
Null handle literal.
or
Alternative representation of the || (logical inclusive or) operator.
true
True boolean literal.
uint
32-bit unsigned integer type.
uint8
uint16
uint32
uint64
n-bit unsigned integer type.
void
Void specifier, used in function definition in place of a returned value type if the function doesn't return any value.
while
while loop declaration.
do-while loop terminating condition declaration.
xor
Alternative representation of the ^^ (logical exclusive or) operator.
